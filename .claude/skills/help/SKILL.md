## Boostability Executive Assistant - Available Skills

---

### Accelerator Tracker

Track and manage all accelerator programs.

| What you want | Say this |
|---|---|
| See all programs and statuses | "show me all the programs I am working on" |
| See only open / not yet applied | "what accelerators are still open?" |
| See what you've applied to | "show me what I've applied to" |
| Add a new program by URL | "add this program: [url]" |
| Add a program manually | "add [program name] to the tracker, deadline is [date], funding is [amount]" |
| Update a program's status | "mark [program name] as applied" |
| Check fit for a new program | "check this program for me: [url]" |

---

### Accelerator Application

Draft answers for any accelerator application.

| What you want | Say this |
|---|---|
| Draft a full application | "help me apply to [program name]" |
| Draft with questions pasted | "draft answers for these questions: [paste questions]" |
| Research a program before applying | "research [program name] before I apply" |
| Focus on one question | "draft an answer for: [paste question]" |

---

### We Make Change Application

Fill out Boostability's registration on We Make Change (volunteer platform).

| What you want | Say this |
|---|---|
| Draft the full application | "fill out the We Make Change application" |
| Draft one specific answer | "draft my answer for the skills need question on We Make Change" |
| Review or refine a draft | "review my We Make Change motivation answer" |

---

### Research

Deep research on any topic, data-first with sources.

| What you want | Say this |
|---|---|
| Research a topic | "research [topic]" |
| Get data for a pitch or application | "find stats on college student mental health and academic performance" |
| Competitive research | "research edtech AI competitors to Boostability" |
| Exhaustive mode | "research --deep [topic]" |

---

### General Assistant Actions

These don't need a specific skill - just say it:

| What you want | Say this |
|---|---|
| Update your priorities | "update my current priorities" |
| Log a decision | "log this decision: [decision]" |
| Draft an email | "draft an email to [person] about [topic]" |
| Summarize a session | "summarize today's session" |
| Remember something permanently | "remember that I always want [X]" |

---

For help on a specific skill, say: "help with [skill name]"
